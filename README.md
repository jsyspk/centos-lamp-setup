# Centos LAMP (lnmp) Setup

This is a command line shell script for lamp stack build and configure steup on centos 7.

You can install
  - nginx
  - php-fpm
  - mysql community
  
You can configure
 - firewall
 
You can add additional services 
 - ntpd
 - nano
 
Additional tools available
 - elasticsearch [current v 6.8]
